# 安装使用

1. 将代码 `git clone` 到本地
2. 使用你常用的代码编辑器打开代码（Sublime，Atom，VSCode等）
3. 在代码目录下运行 `yarn dev`(或者 `npm run dev`)
4. 打开微信开发工具查看效果
